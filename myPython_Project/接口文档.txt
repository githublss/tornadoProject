1.图片验证码
描述：用于生成图片验证码

url：/aip/Possport?codeid=123&pcodeid=
method:get
传入参数：
名称  类型  说明
codeid  int 验证码标号
pcodeid int 上一个验证码标号

返回值：
图片的二进制数据

2.发送短信验证码

url：/api/smscode
method:post
传入参数： json
名称  类型  说明  是否必须
mobile  string
image_code_id   string
image_code_text string
{
    "mobile":"13213818296",
    "image_code_id":"222222",
    "image_code_text":"fmac",
}

返回值：
{
    "error":0,
    "errmsg":"ok"
}
